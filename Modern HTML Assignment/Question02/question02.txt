                HTML Entities 

HTML entities are the special characters that cannot be typed easily on keyboard, html entities are special codes or characters that have special meaning entities have some reserve code in html. with the help of entities we can add icons and various type of shapes by their special characters.

So we uses HTML enties for using reserve characters and displaying special characters.

5 commonly uses Entities are :

1. &copy;      Copyrighyt         &#169;
2. &amp;       Ampersand          &#38;
3. &lt;        Less than          &#60;
4. &gt;        Greater than       &#62;
5. &dollar;    Dollar Sign        &#36;
